AntiCheat

AntiCheat helps server admins easily identify and block malicious users by monitoring and analyzing the behaviour of their players.

Installation

  1. Download and execute the ClientDetector.exe.
  2. You're done! Optionally, explore the configuration and customize to your liking.

Compiling

  1. Ensure you have the JDK, Git, and Maven3 installed.
  2. $ git clone git@github.com:h31ix/AntiCheat.git
  3. $ cd AntiCheat
  4. $ mvn clean install
